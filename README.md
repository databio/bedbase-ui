# BEDbase UI

Web interface for [BEDbase](https://bedbase.org) â€” the open access platform for accessing, aggregating, and analyzing genomic region data.

## Stack

- [React](https://react.dev) + TypeScript
- [Vite](https://vite.dev) for dev/build
- [Tailwind CSS v4](https://tailwindcss.com) + [DaisyUI v5](https://daisyui.com) for styling
- [React Router](https://reactrouter.com) for client-side routing
- [Lucide](https://lucide.dev) for icons
- [highlight.js](https://highlightjs.org) for code syntax highlighting

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
